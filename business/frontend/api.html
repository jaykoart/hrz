API Documentation

Welcome to the Horizon Data API. Use our RESTful endpoints to route HTTP requests through our residential IP network.

Authentication

All API requests require an API key passed via the Authorization header. 

# Header format
Authorization: Bearer YOUR_API_KEY

To obtain an API key, contact our sales team.

Quickstart

Make your first proxied request in seconds:

# cURL example
curl -X POST https://api.horizon-data.io/v1/proxy/request \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "targetUrl": "https://example.com",
    "method": "GET"
  }'

Proxy Request

Send a single HTTP request through a residential IP.

POST /v1/proxy/request

Request Body

Parameter Type Description
targetUrl required string The URL you want to fetch through the proxy.
method string HTTP method (GET, POST, PUT, DELETE). Default: GET
headers object Custom headers to include in the request.
body string Request body for POST/PUT requests.
country string ISO 3166-1 alpha-2 country code for geo-targeting (e.g., "US", "KR").
sessionId string Use the same sessionId to maintain a sticky IP for up to 30 minutes.

Response

{
  "success": true,
  "requestId": "req_abc123",
  "statusCode": 200,
  "headers": { ... },
  "body": "...",
  "proxyIp": "203.0.113.45",
  "country": "US",
  "latencyMs": 342
}

Batch Request

Send multiple requests in a single API call for efficient large-scale data collection.

POST /v1/proxy/batch 
# Batch request example
{
  "requests": [
    { "targetUrl": "https://example.com/page1" },
    { "targetUrl": "https://example.com/page2" },
    { "targetUrl": "https://example.com/page3" }
  ],
  "concurrency": 5
}

Session Management

Use sessionId to keep the same IP for multiple sequential requests (e.g., login flows).

# Session sticky example
{
  "targetUrl": "https://example.com/login",
  "sessionId": "my-session-123",
  "sessionTtlMinutes": 30
}

Geo Targeting

Target specific countries or cities for localized data collection.

# Geo targeting example
{
  "targetUrl": "https://example.com",
  "country": "KR",
  "city": "Seoul"
}

Python SDK

import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.horizon-data.io/v1"

def fetch_via_proxy(url, country=None):
    payload = {"targetUrl": url}
    if country:
        payload["country"] = country
    
    response = requests.post(
        f"{BASE_URL}/proxy/request",
        headers={"Authorization": f"Bearer {API_KEY}"},
        json=payload
    )
    return response.json()

# Usage
result = fetch_via_proxy("https://example.com", country="US")
print(result["body"])

Node.js SDK

const axios = require('axios');

const API_KEY = 'YOUR_API_KEY';
const BASE_URL = 'https://api.horizon-data.io/v1';

async function fetchViaProxy(url, country) {
  const response = await axios.post(
    `${BASE_URL}/proxy/request`,
    { targetUrl: url, country },
    { headers: { Authorization: `Bearer ${API_KEY}` } }
  );
  return response.data;
}

// Usage
fetchViaProxy('https://example.com', 'KR')
  .then(data => console.log(data.body));
