# 데이터 자산화 전략 (Data Asset Strategy)

**문서 버전:** 3.0
**최종 업데이트:** 2025년 12월 8일
**핵심 명제:** "파이프(IP)를 파는 회사"에서 **"정제된 데이터 상품을 파는 회사"**로 진화.

---

## 1. 우리가 수집할 데이터 (Data Universe)

### 1.1. 전략적 집중 분야 (Niche Focus)
빅테크가 관심 없거나, 수집하기 어려운 영역에 집중합니다.

| 분야 | 수집 대상 (예시) | 가치 (활용처) |
| :--- | :--- | :--- |
| **🌾 농업/식량 (Agri-Food)** | 글로벌 곡물 선물 시세, 각국 도매시장 가격, 기후/작황 뉴스 | 농산물 가격 예측 AI, 투자사 리포트 |
| **🌿 환경/기후 (Climate)** | 각국 환경부 공시, 탄소 배출권 거래가, 대기질 센서 데이터 | ESG 보고서 자동화, 기후 리스크 분석 |
| **🛒 이커머스 (E-commerce)** | 글로벌 마켓플레이스 상품 가격, 리뷰, 재고 현황 | 시장 조사, 다이내믹 프라이싱 AI |

### 1.2. 수집 기술 (How We Collect)
*   **Earth VPN 망:** 전 세계 가정 IP를 통해 지역 제한 없이 로컬 데이터 수집.
*   **Headscale 제어:** 특정 국가/도시의 노드만 지정하여 고품질 데이터 확보.
*   **스케줄러:** Cron + Scrapy/Playwright 조합으로 자동 주기 수집.

---

## 2. 데이터 가공 파이프라인 (Processing Pipeline)

```
[Raw Crawl] -> [Cleaning] -> [Deduplication] -> [Anonymization] -> [Labeling] -> [Dataset]
```

### 2.1. 단계별 설명
1.  **Cleaning:** HTML 태그, 광고, 관련 없는 텍스트 제거.
2.  **Deduplication:** 중복 데이터 제거. (MinHash/SimHash 알고리즘)
3.  **Anonymization (익명화):** 이메일, 전화번호 등 PII(개인식별정보) 마스킹. (RegEx + NER 모델)
4.  **Labeling (라벨링):** AI 학습을 위해 정답 태그 부착. (예: 이 뉴스는 "가격 상승 예고" 기사)

---

## 3. 데이터 상품 목록 및 가격표 (Product Catalog)

### 3.1. Raw Data Feed (원본 데이터)
*   **형태:** 정제 전 HTML/JSON 원본.
*   **용도:** 고객이 직접 가공하고 싶을 때.
*   **가격:** GB당 $3 ~ $5 (시장가 대비 50% 저렴).

### 3.2. Curated Dataset (정제 데이터셋)
*   **형태:** CSV/Parquet 형식으로 구조화된 데이터.
*   **예시 상품:**
    *   `Global Grain Price Index (2024-2025)` - $500 / 1회 구매
    *   `Korea KOSDAQ News Sentiment Daily` - $200 / 월 구독
*   **프리미엄:** 빅테크에게도 없는 '우리만의' 데이터. 부르는 게 값.

### 3.3. Labeled Training Data (AI 학습용 데이터)
*   **형태:** 정답 라벨이 붙은 이미지/텍스트 셋.
*   **가격:** 라벨 1개당 $0.05 ~ $0.50 (복잡도에 따라).
*   **서비스:** 고객이 원하는 주제로 맞춤 수집 + 라벨링.
