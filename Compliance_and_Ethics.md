# 데이터 윤리 및 컴플라이언스 전략 (Compliance & Ethics)

**문서 버전:** 3.0
**최종 업데이트:** 2025년 12월 8일
**핵심 명제:** **"세계에서 가장 윤리적인 데이터 공급망"**을 구축하여 빅테크의 신뢰를 얻는다.

---

## 1. 왜 컴플라이언스가 생존의 핵심인가?

*   **고객의 질문:** "당신들 데이터, 합법적으로 수집한 거 맞죠?"
*   **잘못된 대답 시 결과:**
    *   EU 법인: GDPR 위반으로 전 세계 매출의 4% 벌금.
    *   US 법인: CCPA 소송.
    *   빅테크 고객: 계약 해지 및 블랙리스트 등재.
*   **우리의 해자(Moat):** 경쟁사(Bright Data 등)는 합법성 논란에 자주 휘말림. 우리는 **"동의 기반 수집(Consent-Based Collection)"**을 1등으로 증명하면 승리.

---

## 2. 글로벌 규제 체계 (Regulatory Landscape)

| 규제 | 적용 지역 | 핵심 요건 | 우리의 대응 |
| :--- | :--- | :--- | :--- |
| **GDPR** | EU | 명시적 동의, 삭제권, 데이터 이동권 | VPN 가입 시 GDPR 동의 화면, 탈퇴 시 데이터 즉시 파기 |
| **CCPA** | 미국 캘리포니아 | 수집 데이터 공개, 판매 거부권 | "내 데이터가 어떻게 쓰이나요?" 페이지 필수 구현 |
| **PIPA** | 한국 | 개인정보 수집/이용 동의, 제3자 제공 동의 | 약관에 "대역폭 공유 및 데이터 전송 통로 활용 동의" 명시 |

## 3. 동의 관리 시스템 (Consent Management Platform, CMP)

### 3.1. 유저 가입 플로우 (User Journey)
1.  **Earth VPN 앱 설치**
2.  **STEP 1:** "이 앱은 당신의 유휴 네트워크 자원을 공유합니다." (Y/N) -> **명시적 동의 O**
3.  **STEP 2:** "수집된 데이터는 AI 학습 및 연구 목적으로 활용됩니다." (상세 보기 링크) -> **목적 고지 O**
4.  **STEP 3:** 체크박스 + 서명(터치) -> **증거 저장 O**

### 3.2. 동의 기록 저장 (Consent Ledger)
*   모든 동의 기록은 **블록체인(또는 Immutable Log)**에 저장.
*   저장 내용: `{유저ID, 동의버전, 타임스탬프, IP해시}`
*   **감사(Audit)** 시 제출 가능한 법적 증거로 기능.

## 4. 데이터 출처 인증 (Data Provenance Certificate)

### 4.1. 개념
*   데이터셋을 판매할 때, **"이 데이터가 어디서 왔는지"**를 증명하는 인증서를 함께 발급.
*   구매 기업의 법무팀이 안심하고 사용할 수 있게 됨.

### 4.2. 인증서 내용 (예시)
```json
{
  "dataset_name": "Korea Agricultural Price Index Q4 2025",
  "collection_period": "2025-10-01 to 2025-12-31",
  "source_nodes": 12450,
  "consent_coverage": "100% (All nodes opted-in)",
  "pii_status": "Fully Anonymized",
  "issuer": "Earth AI Compliance Engine",
  "certificate_hash": "0xabc123..."
}
```

## 5. 개인정보 비식별화 프로세스 (Anonymization)

### 5. 1. 수집 데이터 중 PII 처리
| PII 유형 | 처리 방법 |
| :--- | :--- |
| 이메일 주소 | `[EMAIL_REDACTED]` 로 치환 |
| 전화번호 | `[PHONE_REDACTED]` 로 치환 |
| 사람 이름 | NER 모델로 탐지 후 `[NAME]` 으로 치환 |
| IP 주소 | 해싱(SHA-256) 처리 (복원 불가) |

### 5.2. 감사 로그 (Audit Log)
모든 데이터 처리 활동은 기록되어 추후 규제 당국의 요청에 대응 가능.
